# Plataforma de Caracterización Sociodemográfica

## Descripción

Proyecto académico desarrollado como parte de la asignatura de Big Data, enfocado en el diseño e implementación de una base de datos relacional para el almacenamiento y gestión de información sociodemográfica.

La solución fue implementada en Microsoft SQL Server y permite representar información relacionada con individuos, ubicación geográfica, educación, ocupación, ingresos, vulnerabilidad, beneficios sociales e intereses mediante un modelo relacional estructurado.

## Objetivos

* Diseñar una base de datos normalizada para la gestión de información sociodemográfica.
* Implementar relaciones uno a muchos y muchos a muchos mediante claves primarias y foráneas.
* Garantizar la integridad referencial de los datos.
* Generar datos sintéticos para validar el funcionamiento del modelo.
* Facilitar consultas y análisis posteriores sobre la información almacenada.

## Tecnologías utilizadas

* Microsoft SQL Server
* SQL
* Modelado Entidad-Relación
* Diseño de Bases de Datos Relacionales

## Características del proyecto

* Base de datos compuesta por 46 tablas.
* Uso de claves primarias y foráneas.
* Implementación de relaciones N:M mediante tablas puente.
* Datos sintéticos para pruebas y validación.
* Modelo orientado a la caracterización sociodemográfica de usuarios.

## Modelo de Datos

El modelo contempla entidades relacionadas con:

* Individuos
* Educación
* Ingresos
* Ocupaciones
* Ubicación geográfica
* Vulnerabilidad
* Beneficios
* Intereses
* Canales de comunicación
